# ⚽ Premier League Analysis (2022–23)

## 📊 Project Overview
This project analyzes Premier League match data using Python and Power BI to derive insights on team performance, points, and goal differences.

---

## 🔧 Tools Used

- Python (Pandas)
  
- Power BI
  
- Data Cleaning & Transformation
  
- Data Visualization

---

## 📁 Data Processing

- Extracted match data from JSON
  
- Cleaned and transformed nested data
  
- Derived features like:
  
  - Goals scored
    
  - Match results
    
  - Points per team
    
- Aggregated data to build league table

---

## 📈 Dashboard Insights

- Arsenal leads the league with highest points
  
- Manchester City shows strongest goal difference
  
- Lower-ranked teams consistently show negative goal differences
